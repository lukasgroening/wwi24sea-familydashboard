# Family Dashboard – WWI24SEA

## Setup

### 1. Repository klonen

```bash
git clone https://github.com/lukasgroening/wwi24sea-familydashboard.git
cd wwi24sea-familydashboard/frontend
```

### 2. Dependencies installieren

```bash
npm install
```

### 3. Projekt ausführen

```bash
npm run dev
```
